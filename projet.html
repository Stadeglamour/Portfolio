M2L

Description :
Ce projet à pour objectif de réaliser une interface graphique en java sur le sujet du sport avec des compétitions, composé de différentes équipes et différents candidats, avec plusieurs possibilités pour chaque (ex: ajout, suppression, modification…). De plus, nous devons relier tout ça sur une base de données.

Langages : Java, JavaFx, MySQL, ...
